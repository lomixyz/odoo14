GOSI Module

Features
  • Calculate GOSI for Saudi employees and add it to the pays lip.
  • Calculate GOSI for non-Saudi employees and add it to the pay slip.
Configuration

1- From payroll module, navigate to configuration, then clicks on “Structures”



2- Navigate to one of the created structures,
Salary structures for Saudi Workers:



Usage

1- To check all employees that have been register inside the GOSI,
Go to Employees -> GOSI Register.



2- To check the details of the GOSI for any employee, just click on the employee name, then details will be appeared in new form.



3- GOSI is detected from the employee’s salary once the pay slip is generated to the employee

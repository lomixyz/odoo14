All in One Invoice reports odoo app helps users to print invoice day wise reports, payment report for invoice, product sales summary reports, invoice report by salesperson report, highest selling products report, top customer product report, invoice day book report with particular date range for particular company in XLS and PDF format. Users can also print excel reports for single or multiple invoices.

Access Rights Configurations

Users with "Show Invoice Custom report" access rights can only print all invoice report, If user wants to see salesperson on invoice payment report user must have "Show Salesperson field in Invoice Payment Report" access.

Day Wise Invoice Report

Day Wise Invoice Report Menu

Under Invoicing > Invoice Custom Report user can see "Day Wise Invoice Report" menu.

On clicking "Day Wise Invoice Report" menu wizard will open here user can select "Start Date" and "End Date", user can also select "Companies" and print report.

Day Wise Invoice PDF Report

On clicking "PRINT PDF" button user can see printed report in PDF format.

Day Wise Invoice XLS Report

On clicking "PRINT XLS" button user can see generated report in XLS format.

Top Customer Product Report

Top Customer Product Report Menu

Under Invoicing > Invoice Custom Report user can see "Top Customer Product Report" menu.

On clicking "Top Customer Product Report" menu wizard will open here user can select "Report Type" as Basic or Compare, If user selects basic report type then enter "From Date", "To Date", "Number of Items", "Total Sale Amount" and "Companies" and print report.

If user selects compare report type then enter "From Date", "To Date", "Compare From Date", "Compare To Date", "Number of Items", "Total Sales Amount" and "Companies" and print report.

Top Customer Product PDF Report

On clicking "PRINT PDF" button user can see printed report in PDF format.

Top Customer Product XLS Report

On clicking "PRINT XLS" button user can see generated report in XLS format.

Invoice Excel Report Menu

User can print report for multiple or single invoice by clicking "Generate Invoice XLS Report" menu under "Action" menu.

If User selects multiple invoice then user can see all invoice report per sheet.

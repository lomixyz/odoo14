Payslip Batch PDF/Excel Report.

Key Features:

  • Allows user to print Payslip Batch PDF & Excel report.
  • User can see the salary computation group by the Salary rule Category & Salary Rules.

Overview

  • From Payslips > Batches menu, From Action select the option Batch Payslip Report.
  • In Opened window, User have an option to print either Excel or PDF report.
  • After click Print Excel button, User can download the Generated Excel report by clicking on the filename.
  • Payslip details into Excel report with group by Salary rule Category.
  • Payslip details into PDF report with group by Salary rule Category.

About
============
Using this stock valuation report module, you can simplify inventory management. This report shows the inventory valuation as selected time and date with the warehouse and company. Reports can be generated based on categories and products. You can choose PDF or XLS formats for analysis.

Installation
============
1) Copy module files to addon folder.
2) Restart odoo service (sudo service odoo-server restart).
3) Go to your odoo instance and open apps (make sure to activate debug mode).
4) click on update app list.
5) search module name and hit install button.

Mail Messages Group

This is an optional extension for Mail Messages Easy

By default all employees have access to "Messages" menu. After installing this addon only users of "Messages: Access "Messages" Menu" will have access to "Messages" menu.

Warning! Restricting access to menu does not restrict access to data itself!

Description

Odoo provides feature of synchronization with Google Calendar, but it limits it mode of meeting i.e. Google Meet. This module adds feature of creating meeting with Google Meet as mode of meeting in Google Calendar. This module also adds feature of connecting Google Meet directly from Odoo.

Features List

  • This module compatible with Google G Suite Account.
  • Events are created for Meet.
  • Connect in meeting using meeting code.

Configuration

For Google Calender configuration follow the steps Click Here

Create Meeting

When user will create a new meeting in Odoo and if Add Hangout Meeting is set on meeting, automatically event is added as Hangout Meeting in Google calender. Invitation mail is send to attendees.

Connect to Meet

When User clicks on icon, user will redirect on Meet page and also connect using meet code.

All in one Receipt Reports - Sales, Purchase, Accounting, Inventory

Currenlty in odoo we can't print reports in receipt printer, This module useful to print reports as receipt. So you can now easily print many odoo standard reports in receipt printer. You don't need to switch to other printer.

  • Very useful to print receipt of sale quotation, sale order, request for quotation(RFQ), purchase order, invoices, bills, delivery order, picking operations, Payment Receipt
  • Quotation/Order Receipt, Purchase Order Receipt, Request For Quotation Receipt, Delivery Slip Receipt, Picking Operations Receipt, Invoices Without Payment Receipt, Invoices Receipt, Payment Receipt Slip.
